# Standing Lifecycle

**Constitution Version:** v5.0

## States

`Proposed`, `Pending Validation`, `Active`, `Suspended`, `Revoked`, and `Retired` are the only valid standing states.

## Allowed transitions

| From | Allowed To |
|---|---|
| Proposed | Pending Validation |
| Pending Validation | Active; Revoked |
| Active | Suspended; Revoked; Retired |
| Suspended | Active; Revoked; Retired |
| Revoked | Retired |
| Retired | None |

`Revoked` and `Retired` standing may never be reactivated. Suspension is temporary but confers no participation right while effective. Every transition requires traceable authority and a ratified Type B or Type C amendment.
