# Audit Constitution

**Constitution Version:** v1.0

## Preamble

Audit is the governed constitutional process through which the entirety of the constitutional system is evaluated for consistency, integrity, traceability, coverage, coherence, and ratification readiness. Audit legitimacy is distinct from authority, capability, policy, standing, claim, trust, verification, reputation, attestation, consensus, governance, voting, federation, economics, and runtime legitimacy. An audit determination may influence ratification readiness only after complete domain coverage, verified integrity across all constitutional dimensions, certified findings, and a valid lifecycle transition to `Certified`.

## Constitutional Doctrine

**Audit ≠ Governance.** Audit evaluates constitutional legitimacy; governance produces collective direction. **Audit ≠ Runtime.** Audit inspects execution artifacts; runtime produces observable action. **Coverage ≠ Certification.** Scanning every artifact does not establish constitutional integrity. **Integrity ≠ Correctness.** Constitutional integrity verifies structure and traceability, not semantic validity. **Audit ≠ Ratification.** A certified audit report is a necessary but not sufficient condition for ratification readiness.
