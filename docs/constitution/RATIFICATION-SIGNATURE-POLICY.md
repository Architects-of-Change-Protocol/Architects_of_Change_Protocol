# Ratification Signature Policy

**Constitution Version:** v1.0

## Signature Requirements

Every ratification signature must include:

- Signature ID
- Signer
- Signer Authority
- Constitution Version
- Ratification Decision
- Timestamp
- Integrity Reference
- Status

## Governing Rule

Signature records constitutional adoption. Signature does not create unilateral authority.
