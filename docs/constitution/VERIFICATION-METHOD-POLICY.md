# Verification Method Policy

**Constitution Version:** v16.0

## Constitutional rule

Every verification type must use at least one authorized method from this catalog. Methods are permanent and unique. A verification performed without an authorized method is constitutionally void.

## Verification method catalog

| Method ID | Method Name | Description | Applies To Classes | Amendment | Status |
|---|---|---|---|---|---|
| VMP-0001 | Constitutional Review | Independent evaluation of constitutional document integrity and version parity | Constitutional | AOC-AMD-0008 | Active |
| VMP-0002 | Document Review | Structured review of governance artifacts against constitutional requirements | Constitutional; Governance | AOC-AMD-0008 | Active |
| VMP-0003 | Witness Attestation | Attested confirmation by an authorized and independent witness with standing | Governance; Runtime | AOC-AMD-0008 | Active |
| VMP-0004 | Cross-System Correlation | Correlation of evidence across independent system records | Runtime | AOC-AMD-0008 | Active |
| VMP-0005 | Independent Audit | Structured audit by an independent authority with operational standing | Operational | AOC-AMD-0008 | Active |
| VMP-0006 | Cryptographic Validation | Cryptographic proof of integrity, origin, or non-repudiation | Operational | AOC-AMD-0008 | Active |

## Method authorization rules

Every verification authority must reference a method ID from this catalog. A verification instance must record the method ID used. No method may be applied outside its authorized classes without a ratified amendment.
