# Federation Capability Policy

**Constitution Version:** v18.0

## Purpose

Capability policy governs how capabilities may be shared, borrowed, federated, or recognized between sovereign constitutional systems. Capability sharing does not transfer ownership.

## Capability sharing modes

| Mode | Description |
|---|---|
| Capability Sharing | A sovereign system makes a capability available to a federated sovereign system within defined scope |
| Capability Borrowing | A sovereign system temporarily exercises a capability owned by another sovereign system within defined scope |
| Capability Federation | Two sovereign systems establish mutual capability recognition within bounded scope |
| Capability Recognition | A sovereign system acknowledges that another sovereign system possesses a specific capability |
