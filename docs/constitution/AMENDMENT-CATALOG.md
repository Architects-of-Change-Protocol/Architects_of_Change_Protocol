## AOC-AMD-0001 — Establish Constitutional Amendment System

- **Amendment ID:** AOC-AMD-0001
- **Title:** Establish Constitutional Amendment System
- **Author:** Architects of Change maintainers
- **Date:** 2026-06-08
- **Type:** Type C
- **Version:** v1.0
- **Affected Laws:** LAW-008
- **Affected Authorities:** Constitution
- **Rationale:** Establish governance of constitutional evolution so future law, authority, ownership, compatibility, composition, and policy changes are versioned, auditable, amendable, and traceable.
- **Risk Assessment:** Low implementation risk. The amendment introduces governance artifacts and scanners without changing existing laws, adding ownership domains, removing transitional authorities, or refactoring Protocol or Enterprise.

## AOC-AMD-0002 — Establish Constitutional Runtime Capability Governance

- **Amendment ID:** AOC-AMD-0002
- **Title:** Establish Constitutional Runtime Capability Governance
- **Author:** Architects of Change maintainers
- **Date:** 2026-06-08
- **Type:** Type C
- **Version:** v2.0
- **Affected Laws:** LAW-008; capability lifecycle, delegation, revocation, and enforcement obligations
- **Affected Authorities:** Constitution; Protocol; Enterprise; Capability Authorities CAP-0001 through CAP-0028
- **Rationale:** Close the constitutional gap between authority existence and authority power by defining, cataloging, assigning, delegating, suspending, revoking, retiring, versioning, and enforcing capability authority.
- **Risk Assessment:** Moderate governance risk and low runtime risk. The amendment introduces fail-closed repository governance but does not implement runtime authorization, permissions, execution, policy, federation, standing, or governance engines.
- **Migration Impact:** Future capability definitions, classes, ownership changes, assignments, delegation rules, revocation rules, and lifecycle changes require Type B or Type C amendment governance. CI and release validation now execute four capability governance scanners.
- **Ratification Status:** Ratified
