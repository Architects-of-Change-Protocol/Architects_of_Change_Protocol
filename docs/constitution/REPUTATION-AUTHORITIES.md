# Reputation Authorities

**Constitution Version:** v12.0

## Constitutional rule

Only reputation types in this catalog may provide governed historical reliability signals to decisions. Reputation IDs are permanent and unique. Catalog presence defines a legitimate reputation type, not an active reputation instance. Retirement preserves every historical record and never reassigns an ID.

## Catalog integrity

Owners must be constitutional authorities. Each source policy, calculation policy, and decay policy must be cataloged. Creation, semantic expansion, ownership transfer, policy changes, and retirement require a ratified Type B or Type C amendment.
