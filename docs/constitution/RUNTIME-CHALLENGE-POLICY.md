# Runtime Challenge Policy

**Constitution Version:** v1.0

## Valid challenge grounds

- `Invalid Authority` — The execution lacked valid constitutional authority
- `Invalid Capability` — The execution invoked a capability outside its authorized scope
- `Invalid Policy` — The execution violated applicable policy constraints
- `Invalid Execution` — The execution deviated from its authorized scope or window
- `Invalid Evidence` — The execution evidence is incomplete, inauthentic, or integrity-compromised
- `Integrity Failure` — One or more integrity dimensions failed
- `Compliance Failure` — One or more compliance obligations were not fulfilled
- `Audit Failure` — The execution cannot be independently audited
- `Economic Failure` — The execution exceeded or violated economic consumption or settlement obligations
