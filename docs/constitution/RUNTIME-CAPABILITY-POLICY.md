# Runtime Capability Policy

**Constitution Version:** v18.0

## Constitutional Rule

Runtime execution may not exceed capability scope.

## Capability invocation dimensions

**Capability Invocation** — The authorized act of activating a constitutional capability within a runtime execution.

**Capability Limits** — Bounded scope, duration, and consumption constraints that a runtime execution may not exceed.

**Capability Dependencies** — Constitutional prerequisites that must be satisfied before capability invocation is valid.

**Capability Consumption** — The measured utilization of a capability during execution, governed by economic consumption policy.

**Capability Completion** — The governed conclusion of a capability invocation, producing evidence of completion.
