# Trust Authorities

**Constitution Version:** v10.0

## Constitutional rule

Only trust types in this catalog may provide confidence to governed decisions. Trust IDs are permanent and unique. Catalog presence defines a legitimate trust type, not an active trust instance. Retirement preserves every historical record and never reassigns an ID.

## Catalog integrity

Owners must be constitutional authorities. Each evidence policy covers exactly one trust type. Creation, semantic expansion, ownership transfer, decay or revocation changes, and retirement require a ratified Type B or Type C amendment.
