# Federation Governance Policy

**Constitution Version:** v1.0

## Purpose

Federated governance governs how sovereign constitutional systems coordinate governance acts across federation boundaries. Federated governance cannot override local sovereignty.

## Federated governance constructs

| Construct | Description |
|---|---|
| Federated Governance Motion | A governance motion that requires participation from multiple sovereign constitutional systems |
| Federated Governance Consensus | A consensus determination that spans sovereign constitutional system boundaries |
| Federated Governance Mandate | A governance mandate that applies within federated scope across participating sovereign systems |
| Federated Governance Outcome | A governance outcome that records the result of a federated governance act |
