## Consolidation policy

- `@aoc/protocol/contracts` owns actor, capability, delegation, consent, policy, decision, and audit contract shapes.
- `@aoc/protocol/claims` owns capability, delegation, grant, verification, trust, attestation, credential, and revocation claim vocabulary.
- `@aoc/protocol/errors` owns access, verification, authorization, trust, revocation, execution, governance, policy, and runtime error contracts.
- `@aoc/protocol/adapters` owns all external lookup, decision-provider, authorization-provider, event-sink, and legacy port seams.
- Legacy exports remain available; no implementation or consumer source is moved by this consolidation.
